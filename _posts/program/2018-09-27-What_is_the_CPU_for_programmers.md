# 前言

* 程序是什么？
  *  指示计算机每一步动作的一组指令和数据
  * 譬如运动会、音乐会的程序，指的是`行事的先后次序`
* 正在运行的程序存储在什么位置？
  * 内存
  * 硬盘等媒介上保存的程序被复制到内存后才能运行
* 什么是内存地址？
  * 内存中用来表示命令和数据存储位置的数值
* 什么是机器语言
  * CPU 可以直接识别并使用的语言
* 计算机的构成元件中，负责程序的解释和运行的是哪个？
  * cpu
